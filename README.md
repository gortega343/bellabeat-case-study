# Bellabeat Case Study — Análisis de Wearables

## Descripción
Análisis de datos de dispositivos FitBit para identificar tendencias de comportamiento 
en usuarios de wearables. Proyecto desarrollado como capstone del 
Google Data Analytics Certificate.

## Herramientas utilizadas
- Google Sheets — limpieza y análisis de datos
- Tableau Public — visualización de resultados

## Hallazgos principales
- El 78% de los usuarios no alcanza los 10,000 pasos diarios recomendados por la OMS
- El 81% del tiempo registrado corresponde a comportamiento sedentario
- Los usuarios que duermen entre 6 y 8 horas queman más calorías (2,527 cal/día)
- El 88% del tiempo activo es actividad ligera
